# Warframe-Market-Script
🚀 Funzionalità Principali
Gestione Sessione: Autenticazione automatica rapida tramite token JWT.

Automazione Sindacati: Gestione completa dei sindacati attivi.

Controllo Prezzi: Impostazione di soglie minime di pubblicazione e timer di aggiornamento.

Vendita Rapida: Interfaccia dedicata per la gestione di singoli oggetti e prezzi in Platinum (PL).

Monitoraggio Reale: Log integrato per vedere cosa fa il bot in tempo reale.

📋 Prerequisiti
Python

Selenium

Google Chrome (rilevamento automatico del percorso)

⚙️ Configurazione
Vai su Warframe.market.

Premi F12 -> Application -> Cookies per recuperare il token JWT.

Salva il token nel file config.txt.

🚀 Avvio
Esegui il file WFMScript.bat per far partire l'interfaccia.

🛑 Gestione Emergenze
Pulsante PANICO: Cancella immediatamente tutti gli ordini attivi sul mercato e blocca ogni operazione in corso.
